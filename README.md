## 📌 Project Overview

Task Automation Script is a Python-based utility designed to automate the process of moving JPG image files from one folder location to another.

The application reduces repetitive manual file management tasks by automatically detecting image files, creating destination folders when required, and transferring files efficiently using Python's built-in libraries.

---

## ✨ Features

- 📷 Automatic JPG file detection
- 📂 Automated file movement between folders
- 📁 Destination folder creation
- ⚡ Fast and efficient execution
- 💻 Simple console-based operation
- 🔄 Reduces repetitive manual work

---

- `os` Module

## 🚀 How To Run

### Step 1: Install Python

Download and install Python 3 from the official website.

### Step 2: Add JPG Files

Place JPG image files inside the source folder.

### Step 3: Run the Script

Open the terminal and execute:

```bash
python move_images.py
```

The script will automatically detect JPG files and move them to the destination folder.
